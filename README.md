# Премиальная версия свадебного сайта

Готовый статический сайт-приглашение для GitHub Pages, Netlify, Vercel или любого обычного хостинга.

## Что сделано

- Премиальный mobile-first UI в мягкой свадебной палитре.
- Hero-блок с SVG-аркой, монограммой, лепестками и плавным появлением.
- Блоки с важными моментами: дата, место, предупреждение про водоём.
- Календарь мая 2026 с выделенным 26 числом.
- Программа дня, дресс-код, локация, второй день.
- Анкета гостя с сохранением в localStorage и кнопкой копирования ответа.
- Плавные scroll-анимации через IntersectionObserver.
- SVG-иконки без внешних библиотек.
- Код без сборщиков и зависимостей: достаточно открыть `index.html`.

## Как поменять данные

Основные настройки находятся в начале `app.js`:

```js
const CONFIG = {
  couple: ['Артур', 'Алиса'],
  dateISO: '2026-05-26T16:00:00+03:00',
  weddingDay: 26,
  calendarMonthIndex: 4,
  calendarYear: 2026,
};
```

Текстовые блоки можно редактировать прямо в `index.html`.

## Музыка

Кнопка музыки уже добавлена. Чтобы включить трек, положите MP3 в папку `assets`, например `assets/music.mp3`, и в `index.html` замените пустой `src`:

```html
<source src="assets/music.mp3" type="audio/mpeg" />
```

## Публикация на GitHub Pages

1. Создайте репозиторий.
2. Загрузите файлы из архива в корень репозитория.
3. Откройте Settings → Pages.
4. Source: Deploy from a branch → `main` → `/root`.
5. Сохраните — сайт появится по ссылке GitHub Pages.
